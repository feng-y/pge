Context Friction Notes

Future agents often start at the route and miss that webhook reconciliation owns the durable state. Start at packages/billing/subscriptions.ts before editing UI copy or plan selection logic.

Do not infer subscription state from Stripe checkout session status; use local subscription rows after webhook reconciliation.
