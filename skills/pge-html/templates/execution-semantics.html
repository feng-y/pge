Semantic Model

One request path performs multiple feature executions, stores intermediate feature maps, then expands them into the final model batch shape. Read this as runtime semantics, not just a module map.

Agent Gotchas First

Name mismatch
Some functions named like tensor builders may only prepare maps; confirm where tensors are actually created.
Shared context
Later execution may depend on state initialized earlier. Do not isolate a branch without checking context setup.
Different inputs
The same runner may consume different semantic fields in each branch. Verify which pointer or state field drives the slot logic.
